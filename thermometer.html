TURRIS:GADGETS thermometer


This is a demo page for a thermometer using the TURRIS:DONGLE and a wireless thermometer TP-82N. It displays data from a file thermometer.csv which should be created by the script thermometer.py which does the actual communication with the thermometer.

If you do not see any data here, the code is either not running or did not produce any data yet. Check that the file thermometer.csv was created by the script and that it contains some data. Also your browser might refuse to load the data from the local storage - while Firefox permits this, Chrome does not.

This page uses Twitter Bootstrap for styling and Dygraphs for the chart.
